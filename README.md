# SCT_ML_2

## Customer Segmentation using K-Means Clustering

### Objective
The objective of this project is to group customers of a retail store into different segments based on their characteristics and purchasing behavior using the K-Means Clustering algorithm.

### Dataset
Mall Customer Dataset

### Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn

### Steps Performed
- Loaded and explored the dataset
- Checked for missing values
- Analyzed statistical properties
- Selected relevant features
- Standardized the data
- Determined the optimal number of clusters using the Elbow Method
- Applied K-Means Clustering
- Visualized customer segments
- Analyzed cluster profiles
- Saved the clustered dataset

### Cluster Insights
- Cluster 0: Low income, low spending customers
- Cluster 1: Young customers with moderate income and spending
- Cluster 2: High income, high spending customers
- Cluster 3: High income, low spending customers
- Cluster 4: Older customers with moderate income and average spending

### Outcome
The model successfully segmented customers into different groups, helping businesses better understand customer behavior and design targeted marketing strategies.
